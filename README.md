# Secure Client-Server Authentication System 🔐

A comprehensive security project demonstrating a full authentication flow between a **Python** client and a **Node.js** backend. Developed as part of a structured software development lifecycle.

### 🚀 Project Overview
This system implements a secure "Login & Sign-up" process, focusing on data integrity and encrypted communication. The project was managed using **Agile methodology** and divided into strategic milestones (Sprints).

### 🛠 Key Technical Features
* **Multi-Language Architecture:** Seamless integration between a **Python** frontend/logic layer and a **Node.js** server.
* **Advanced Cryptography:**
    * **AES Encryption:** Ensuring secure data transfer between client and server.
    * **Hashing (SHA-256):** Secure password storage and verification.
* **Database Management:** Custom scripts (`getData_DB.py`) for handling persistent user data and authentication records.
* **Network Communication:** Robust request-response handling using modern backend protocols.

---

### 📅 Development Milestones (Sprints)
1. **Sprint 1:** Foundation of encryption logic and basic client-side Python implementation.
2. **Sprint 2:** Implementation of complex authentication flows and session management.
3. **Node Server Integration:** Developing the JavaScript-based backend to handle concurrent requests and DB connectivity.
